Welcome to text processing server !

Following are the APIs :

API Function Example
/getKeywordsBatch/ (POST method) Get keywords and phrases with their ranks and frequencies from text corpus curl -H "Content-Type: application/json" -X POST -d '{"responses":["love is a drug", "i love excellent service"]}' http://lemmatizer-api.herokuapp.com/getKeywordsBatch/
/getKeywordsWithSentiment/ (POST method) Get keywords and phrases with their ranks and frequencies from text corpus,and retrieve a sentiment score corresponding to each response curl -H "Content-Type: application/json" -X POST -d '{"responses":["sky is blue", "i love excellent service"]}' http://lemmatizer-api.herokuapp.com/getKeywordsWithSentiment/
