Welcome to text processing server !

Following are the APIs :

API Function Example
/getKeywords/ (GET method) Get keywords and phrases with their ranks and frequencies from text corpus https://lemmatizer-api.herokuapp.com/getKeywords/Life is a matter of good choices and bad choices.
/getLemmas/ (GET method) Get lemmas of words which are passed as comma seperated string https://lemmatizer-api.herokuapp.com/getLemmas/loved,rocks,love
/transformCorpus/ (GET method) Tokenize, Remove stop words, Lemmatize the corpus https://lemmatizer-api.herokuapp.com/transformCorpus/(This is a string)
/addFilterWords/ (POST method) Adding filter words curl -H "Content-Type: application/json" -X POST -d '{"words":["love"]}' https://lemmatizer-api.herokuapp.com/addFilterWords/
/removeFilterWords/ (POST method) Remove filter words curl -H "Content-Type: application/json" -X POST -d '{"words":["love"]}' https://lemmatizer-api.herokuapp.com/removeFilterWords/
